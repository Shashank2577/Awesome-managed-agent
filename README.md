# Awesome-managed-agent

An agentic system prototype with runtime lifecycle enforcement, evented execution, and a spec-aligned UI console.

## Run tests

```bash
PYTHONPATH=. pytest -q
```

## Run the UI + API demo

```bash
PYTHONPATH=. python scripts/run_runtime_ui.py
```

Then open: `http://127.0.0.1:8080`
